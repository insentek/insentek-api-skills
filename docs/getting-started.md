# Getting Started — 快速开始

> 5 分钟内配置完成，开始用自然语言查询设备数据。

---

## 你需要什么

| 项目 | 说明 | 获取方式 |
|------|------|---------|
| `appid` | 应用 ID | E 生态后台创建应用后获得 |
| `secret` | 应用密钥 | 与 appid 同时生成 |
| 设备 | 已接入 insentek 平台的物联网设备 | 联系 insentek 或自行部署 |

---

## 第一步：获取认证信息

1. 登录 [E 生态](https://www.ecois.info)
2. 进入「应用管理」→「创建应用」
3. 复制 `appid` 和 `secret`
4. **妥善保存 secret，不要泄露给他人**

---

## 第二步：选择平台并加载 Skill

将 `skill.md` 文件内容加载到你使用的 Agent 平台：

| 平台 | 加载方式 | 详见 |
|------|---------|------|
| **OpenClaw** | 导入 skill 文件 | [platform-setup.md](platform-setup.md#openclaw) |
| **Claude Code** | 放入项目目录或粘贴到对话 | [platform-setup.md](platform-setup.md#claude-code) |
| **ChatGPT** | 粘贴到 Custom Instructions 或 GPTs 配置 | [platform-setup.md](platform-setup.md#chatgpt) |
| **Hermes-Agent** | 导入技能包 | [platform-setup.md](platform-setup.md#hermes-agent) |

---

## 第三步：开始对话

首次对话时，提供你的 `appid` 和 `secret`：

> **User:** 我的 appid 是 test，secret 是 test，帮我查看所有设备。

Agent 会自动：
1. 用 appid + secret 获取 token
2. 查询你的设备列表
3. 返回设备信息

后续对话无需重复提供认证信息（token 会在对话中自动缓存）。

---

## 常用查询示例

### 查看设备列表
```
查看我的设备
```

### 查询实时数据
```
3号设备现在的温度是多少？
```

### 查询历史数据
```
1号设备上周的土壤湿度变化
```

### 多设备对比
```
对比一下1号和2号设备的温度
```

### 异常检测
```
检查一下所有设备的电池状态
```

## 支持的设备类型

| 类型代码 | 设备名称 | 典型参数 |
|---------|---------|---------|
| `Z` | 土壤墒情仪 | 土壤温度、土壤水分、电导率 |
| `T` | 气象站 | 空气温度、湿度、风速、降雨量 |
| `J` | 见厘液位计 | 激光液位、电池电压 |

## 时间表达支持

你可以用自然语言描述时间，Agent 会自动转换：

| 你说 | Agent 理解 |
|------|-----------|
| "现在" / "最新" / "实时" | 调用实时数据接口 |
| "昨天" | 昨天的全部数据 |
| "最近7天" / "近一周" | 过去 7 天的数据 |
| "上周" | 上周一至周日 |
| "本月" | 本月 1 号到今天 |
| "某时刻" / "12点30分" | 指定时间点的数据 |
| *(没提时间)* | 默认最近 24 小时 |

## 常见问题

**Q: token 多久过期？**
A: 默认 2 小时（7200 秒）。Agent 会在过期前 5 分钟自动刷新，无需你操作。

**Q: 可以用设备别名查询吗？**
A: 可以。Agent 支持别名部分匹配，如 "3号" 会匹配别名为 "3号大棚" 的设备。

**Q: 数据查询有时间范围限制吗？**
A: 增量同步接口（/incremental）首次调用返回最近 3 个月数据。历史数据查询最大支持 3 年回溯，单次查询跨度不超过 365 天。建议单次查询不超过 30 天以提高响应速度。

**Q: 支持英文查询吗？**
A: skill.md 中的 Prompt 指令为中文，但 Agent 平台通常支持多语言理解。英文查询的效果取决于具体 Agent 平台。
