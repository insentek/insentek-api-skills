# Examples: Alert Analysis — 异常分析示例

> 展示 Agent 如何检测异常数据并生成告警报告。

---

## Example 1: 单设备异常检测

### User Input
```
检查一下1号大棚的数据有没有异常
```

### Agent Thinking
1. alias="1号大棚" → sn
2. 时间: 无 → 默认最近 24 小时
3. 需要查询所有参数，然后扫描异常规则

### Tool Calls
```
query_device(alias="1号大棚") → sn = "00000000000001"

query_data(sn="00000000000001")
  → GET /v3/device/00000000000001/data?range=20250512,20250513
```

### Formatted Output

```

## Example 2: 温度突变告警

### User Input
```
气象站A今天有没有温度突变
```

### Agent Thinking
1. alias="气象站A" → sn
2. 时间: "今天" → range = today ~ today
3. 参数: "温度" → airTemperature
4. 检查温度突变规则: 1小时内变化 > 10℃

### Tool Calls
```
query_device(alias="气象站A") → sn = "00000000000003"

query_data(sn="00000000000003", time_expression="今天", includeParameters="airTemperature")
  → GET /v3/device/00000000000003/data?range=20250513,20250513&includeParameters=airTemperature
```

### Formatted Output

```

## Example 3: 多设备批量异常扫描

### User Input
```
检查所有设备的电池状态
```

### Agent Thinking
1. 无 alias/sn → 查询所有设备列表
2. 对每个设备查询最新数据中的 battery 参数
3. 应用异常规则: battery < 3.0V → 严重

### Tool Calls
```
query_device(page=1, limit=100)
  → 3 devices: [0001, 0002, 0003]

query_data(sn="00000000000001", time_expression="现在", includeParameters="battery")
query_data(sn="00000000000002", time_expression="现在", includeParameters="battery")
query_data(sn="00000000000003", time_expression="现在", includeParameters="battery")
```
