# Examples: Report Generation — 报告生成示例

> 展示 Agent 如何根据设备数据生成结构化报告。
> 报告生成指南见 [`docs/analysis.md`](../docs/analysis.md)。

---

## Example 1: 设备日报

### User Input
```
给我生成1号大棚昨天的日报
```

### Agent Thinking
1. alias="1号大棚" → 解析 sn
2. 时间: "昨天" → range = yesterday ~ yesterday
3. 需要查询所有参数，生成综合分析报告

### Tool Calls
```
query_device(alias="1号大棚")
  → sn = "00000000000001"
  → GET /v3/device/00000000000001/description

query_data(sn="00000000000001", time_expression="昨天")
  → GET /v3/device/00000000000001/data?range=20250512,20250512
```

## Example 2: 周度对比报告

### User Input
```
对比一下1号大棚本周和上周的土壤数据
```

### Agent Thinking
1. alias="1号大棚" → sn
2. 需要两个时间范围: 本周(this Monday ~ today) + 上周(last Monday ~ last Sunday)
3. 查询土壤相关参数（temperature, moisture, ec）
4. 生成对比分析

### Tool Calls
```
query_device(alias="1号大棚") → sn = "00000000000001"

query_data(sn, time_expression="本周")
  → GET /data?range=20250512,20250513

query_data(sn, time_expression="上周")
  → GET /data?range=20250505,20250511
```
